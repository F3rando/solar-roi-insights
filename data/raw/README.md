# Raw inputs (not committed)

Place **immutable** extracts here before any cleaning (ZenPower CSVs, sponsor drops, Census snippets, etc.).

**Naming:** `YYYY-MM-DD_source_short-name.ext` (example: `2026-04-19_zenpower_installs.csv`).

**Do not commit secrets or PII.** If files must stay local only, keep them here — this directory’s contents are git-ignored except this README.

When you finish a notebook run, record the exact filenames you used in `data/lineage/run_log.json` (see example file there).
